# Simple E-commerce

A small React-based e-commerce frontend project (simple product listing, cart and pages).

## Features
- Product listing pages (men, women, kids)
- Add to cart and cart view
- Contact, FAQ and basic forms

## Prerequisites
- Node.js 14+ and `npm` or `yarn`

## Install

```bash
npm install
```

## Run (development)

```bash
npm start
```

## Build (production)

```bash
npm run build
```

## Project structure
- `public/` — static HTML and manifest
- `src/` — React source files
  - `src/components/` — UI components used by the app

## Notes
- `node_modules` has been removed from the repo and is listed in `.gitignore`. Run `npm install` locally before starting.
- If you want me to purge large files from the repository history (e.g. previously committed `node_modules`), I can run a history rewrite with `git filter-repo` or BFG and force-push.
